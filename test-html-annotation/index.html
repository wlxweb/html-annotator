体验路径

01. 打开悬浮按钮

拖动后再点击，验证拖拽与点击不会互相误触。

02. 进入标注模式

选一个卡片或按钮，输入 Markdown 内容，按 Enter 保存。

03. 查看全部标注

检查列表项布局、图标、动画和定位跳转效果。
